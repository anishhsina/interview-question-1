Inline, Internal, and External Stylesheet

TypeDescriptionExample
Inline Written directly in the element <h1 style="color:red;">Hello</h1>
Internal Inside <style> tag in HTML head <style> h1 { color: red; } </style>
External In separate CSS file <link rel="stylesheet" href="style.css">

What does justify-content do?

Aligns items along the main axis in Flexbox.

Values: flex-start, center, flex-end, space-between, space-around, space-evenly

.container { display: flex; justify-content: center; }

What is z-index in CSS?

Controls the stacking order of positioned elements.

Higher z-index brings element to the front. Only works with position: relative, absolute, or fixed.

.box { position: absolute; z-index: 10; }
